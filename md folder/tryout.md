# Dreamland College Management System Tryout Guide

You can try out the system using the following demo accounts. All accounts use `password123` as the default password (unless otherwise specified).

## Demo Accounts

| Role          | Username      | Password     | Description                          |
|---------------|---------------|--------------|--------------------------------------|
| Superadmin    | admin         | admin123     | Full system access                   |
| Registrar     | registrar     | registrar123 | Student management and registration  |
| Accountant    | accountant    | accountant123| Payment verification                 |
| Faculty       | faculty       | faculty123   | Grade entry and course management    |
| Student       | student       | student123   | Enrollment and personal record access|

## How to use

1.  Open the application in your browser (usually [http://localhost:3000](http://localhost:3000)).
2.  Navigate to the Login page.
3.  Enter the username and password from the table above.
4.  Explore the features available for each role.

## Example Workflow

1.  **Login as Admin** to see the overall dashboard.
2.  **Login as Registrar** to manage students in the "Addis Ababa Main" branch.
3.  **Login as Student** to view your academic status and enroll in available courses.
